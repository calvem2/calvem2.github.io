The American Fire Problem

CSE 442: Data Visualization Project | Nov 2020
HTML, CSS, JavaScript, D3
Team: Megan Calverley, Claire Beard, Anne Pham, Andrew Lee, Ariel Shurygin

For this project, we focused on wildfires in the United States. Using D3, we explored how the number and size of wildfires over time in each state and the U.S. as a whole.
