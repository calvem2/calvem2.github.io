Guess Who?

personal project | March 2021
React, Firebase, PubNub
Team: Megan Calverley, Ash Shah

In an ever virtual world, my friends and I are always looking for new online games to play. When we couldn't find an online version of 'Guess Who?' we decided to make our own. Ash designed our version of the game and we implemented her vision using React. The in-game chat functionality is implemented using PubNub, and the game is deployed using Firebase.
